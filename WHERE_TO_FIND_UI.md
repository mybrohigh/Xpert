# 📍 Где найти новый UI в Xpert Panel

## 🎯 **Быстрый доступ к Traffic Manager**

### 🌐 **Способ 1: Прямая ссылка**
```
http://your-domain.com/#/traffic/
```

### 📱 **Способ 2: Через меню**
1. Откройте Xpert Panel: `http://your-domain.com`
2. Нажмите на **☰** (три полоски) вверху справа
3. В выпадающем меню выберите **"Traffic Manager"**

## 🔧 **Кнопки управления**

### 🔄 **Кнопка "Refresh"**
- **Расположение:** Вверху справа
- **Функция:** Обновить всю статистику
- **Когда использовать:** После добавления нового трафика

### 🗑️ **Кнопка "Reset All"**  
- **Расположение:** Вверху справа
- **Функция:** Полный сброс статистики
- **Внимание:** Удаляет ВСЕ данные!

### 🧹 **Кнопка "Cleanup Old Data"**
- **Расположение:** В разделе Database Information
- **Функция:** Удалить записи старше 90 дней
- **Когда использовать:** Для оптимизации БД

## ⚠️ **Если UI не видно**

### 🔍 **Проверьте что UI запущен:**
```bash
# UI должен работать на порту 3000
curl http://localhost:3000

# Xpert Panel API на порту 8000  
curl http://localhost:8000/api/xpert/stats
```

### 🔄 **Перезапустите сервисы:**
```bash
# Перезапустить Xpert Panel
sudo systemctl restart xpert-panel

# Перезапустить UI (если нужно)
cd /opt/xpert_panel/app/dashboard
npm run dev
```

### 🌐 **Проверьте порты:**
- **UI:** http://localhost:3000
- **API:** http://localhost:8000
- **Nginx:** http://your-domain.com (прокси)

UI адаптивен и работает на:

- 💻 Ноутбуки
- 🖥️ Настольные ПК

## 🎯 **Быстрые действия**

### 📊 **Просмотр общей статистики:**
1. Откройте Traffic Manager
2. Смотрите раздел "Global Statistics"

### 👤 **Просмотр статистики пользователя:**
1. Введите user_token в поле "User Token"
2. Нажмите Enter или подождите авто-обновления
3. Смотрите таблицу с серверами

### 🗑️ **Сброс статистики:**
1. Нажмите "Reset All"
2. Подтвердите в модальном окне
3. Дождитесь завершения

### 🧹 **Очистка старых данных:**
1. Нажмите "Cleanup Old Data"
2. Подтвердите в модальном окне
3. Дождитесь завершения

## 🔗 **Интеграция с Marzban**

### 🎛️ **Кнопки в Marzban UI теперь работают:**
- **"Сбросить трафик"** → Очищает ВЕСЬ трафик
- **"Лимит трафика"** → Учитывает внешний трафик

### 📍 **Где найти в Marzban:**
1. Откройте Marzban Panel
2. Перейдите в раздел администраторов
3. Используйте привычные кнопки управления
